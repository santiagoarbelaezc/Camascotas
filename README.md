<h1 align="center">🐾 Camascotas - E-commerce</h1>

<h3 align="center">🚀 Plataforma en desarrollo para venta de camas y productos para mascotas</h3>

<p align="center">
  Aplicación web full‑stack para e‑commerce especializado en el bienestar y confort de las mascotas.<br>
  Frontend en Angular, Backend en PHP (REST API), con despliegue progresivo en la nube.
</p>

---

### **PENDIENTE PARA EL PROYECTO**
- Integración final de pasarela de pagos (Epayco)
- Optimización de carga de imágenes con Cloudinary (Optimizacion de las imagenes)
- Mejoras en el asistente IA

## 📋 **Descripción del Proyecto**

**Camascotas** es una plataforma de comercio electrónico especializada en camas, accesorios y soluciones para el descanso de mascotas. Con base en Colombia, el proyecto busca ofrecer una experiencia de usuario premium para dueños de mascotas que buscan calidad y diseño.

> ⚠️ **Estado del Proyecto:** En desarrollo activo

---

## 🏗️ **Dashboard**

**Camascotas Dashboard Admin**: se implementará un panel administrativo para gestionar el catálogo de productos, permitiendo la creación, edición y eliminación de artículos, así como la visualización de estadísticas de ventas y vistas de productos.

## 🏪 **Características del E-commerce**

### **🔍 Catálogo de Productos**
- Búsqueda y filtrado de productos por categorías (Camas, Accesorios, etc.).
- Productos aleatorios y destacados en la página principal.
- Incremento de contador de vistas por producto para análisis de popularidad.
- Vista detallada con imágenes gestionadas por Cloudinary.

### **🛒 Gestión de Compras**
Próxima funcionalidad: Implementación de carrito de compras local y persistencia de pedidos para usuarios registrados.

---

## 📦 **Backend**

- API REST construida puramente en PHP.
- Autenticación robusta mediante JWT (JSON Web Tokens).
- Gestión de imágenes integrada con Cloudinary.
- Arquitectura orientada a controladores y middlewares para seguridad.
- Scripts de base de datos incluidos para despliegue rápido.

---

## 🖥️ **Frontend**

- Single‑page application (SPA) con Angular 18+.
- Estilizado moderno y responsive con Tailwind CSS.
- Estructura modular: servicios de autenticación, productos y utilidades.
- Guards de ruta para protección de áreas administrativas.
- Integración con Firebase para hosting y optimización.

---

## 🚀 **Cómo ejecutar el proyecto**

### Backend (PHP)
```bash
cd backend-php
composer install
# configurar variables de entorno en .env
php -S localhost:8000
```

### Frontend (Angular)
```bash
cd frontend
npm install
ng serve
```
